# Intake Checklist

Use this checklist the moment a new manufactured-home foundation certification request arrives.

## 1. Create the case record

- Assign a case ID.
- Record the intake date and target turnaround.
- Log referral source, borrower contact, lender contact, and any appraiser contact.
- Classify the initial product type: `standard_manufactured_home` or `boxabl`.
- Record the property state and note the engineer license needed for the final letter.

## 2. Confirm scope

- Determine whether the request is for a foundation certification letter, site verification only, Boxabl foundation guidance, or a broader engineering scope.
- Confirm whether the lender, appraiser, or AHJ has any special wording or attachment requirements.
- Note any accessory structures already disclosed: porch, deck, stairs, carport, garage, awning, room addition, retaining wall, ramp.

## 3. Log the evidence package

- Appraisal report or equivalent property summary received
- Exterior photo set received
- Foundation photo set received
- Crawlspace or underside photos received, if accessible
- Survey, site plan, plot plan, or sketch received, if available
- Any manufacturer, installer, or prior engineer documents received
- Any notes about movement, re-leveling, settlement, or repairs received

## 4. Initial quality screen

- Photos are current and legible.
- Foundation views show multiple sides.
- Underside visibility is enough to understand support type and anchorage.
- Address in the package is internally consistent.
- The package contains enough information to identify whether additions may affect the load path.

## 5. Routing decision

- If the minimum remote package is complete and no red flags are obvious, send to remote review.
- If the package is incomplete but likely resolvable, send a request for more evidence.
- If visibility or site conditions appear unsuitable for remote verification, open a site-tech dispatch path.

## Intake outputs

- Updated case record
- Evidence inventory with `present`, `missing`, or `not_applicable`
- Product classification
- Initial triage status
- Owner of next action
