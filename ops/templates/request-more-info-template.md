# Request More Information Template

Use this when the case is probably resolvable remotely but the evidence package is incomplete.

## Borrower / client version

Subject: Additional photos and documents needed for foundation certification review

Hello,

We reviewed the submitted package for the manufactured-home foundation certification request. Before we can complete the engineering review, we still need the following items:

- [insert missing item]
- [insert missing item]
- [insert missing item]

Please send clear current photos or documents that address the missing items above. If any area is inaccessible, tell us why and include the best available photos from nearby viewpoints.

Once the missing items are received, we will continue the review and confirm whether the case can remain remote or requires a site visit.

Thank you.

## Internal processor notes

- State exactly what is missing.
- Say whether the issue is a document gap or a visibility gap.
- Give examples of the photo angles needed.
- Avoid implying approval before the engineer completes the review.
