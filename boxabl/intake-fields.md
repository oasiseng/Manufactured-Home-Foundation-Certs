# Boxabl Intake Fields

Log these fields before deciding whether the case can stay on a desk-review path.

## Product identity

- product name and model
- module count and configuration
- revision or package date
- manufacturer foundation package received

## Site and jurisdiction

- project address
- city and state
- permit authority or AHJ
- flood, slope, wind, or seismic concerns disclosed

## Foundation basis

- proposed foundation type
- installer or contractor identified
- site plan, plot plan, or grading sketch received
- utility entry and service coordination notes received

## Accessory scope

- decks
- stairs
- landings
- ramps
- carports or canopies
- retaining features or podium conditions

## Deliverable target

- foundation certification style letter
- foundation guidance memo
- stamped plan review path
- scope clarification only
