# Boxabl Review Checklist

Use this checklist after Boxabl intake is complete.

## Product package review

- Manufacturer foundation package is present.
- Module configuration is identified clearly.
- Submitted field conditions do not obviously conflict with the product basis.

## Site review

- Site plan or equivalent plot information is available.
- Drainage, slope, and access conditions are documented clearly enough for the current scope.
- Utility entry points or clearance constraints are known if they affect the foundation or platform layout.

## Scope review

- It is clear whether the request is a letter, a guidance memo, or a design/stamped-plan path.
- Accessory structures are identified separately from the base product.
- Any custom platform, podium, or elevated condition is documented clearly.

## Escalation triggers

- missing manufacturer package
- unclear module configuration
- custom support geometry not covered by the submitted basis
- deck, stair, or landing loads that appear integrated with the main support path
- field conditions that do not match the assumed product installation basis
